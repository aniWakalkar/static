useState( ) :
  • useState() hook can be only used in the functional component.
  • useState() hook are used to change the state of a component.
  • when a useState() hook changes any state in the component, only that component re-renders.

Syntex :
const [state, newState] = useState( Intial_Value );

  • It returns an array with two values: the current state and a function to update it.
  • state contains the current value.
  • newState is the function using which we can set new value to the state variable.
  • Intial_Value is the default value of our state.

How to use :

Given example is the simplest way to understand that how useState() hook works.

App.js

import {React, useState} from 'react' ;

const app = ( )=>{
const [count, setCount] = useState( 0 );
    return (
        <>
          <div>
          <h1> Count - { count } </h1>
          <button onClick={ ()=> setCount(count + 1) }> IncreaseCount </button>
          </div>
        </>
    )}


Notice :

      setCount(count + 1)     Not a good way to update a state using it's previous state , we are using this way for better understanding.
